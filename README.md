# 🌿 BC Mental Health Resources

A warm, accessible directory helping people in British Columbia find mental health support in their community.

## Features

- **Regional District selector** — choose from all 29 BC regional districts to find local services
- **10 resource categories** — Crisis & Emergency, Government Services, Non-Profits, Medical & Clinical, Support Groups, Indigenous-Specific Services, Youth & Family, Virtual & Online, Substance Use, and Workplace & Financial
- **65+ resources** — crisis lines, health authority services, CMHA branches, Foundry centres, and more
- **Search** — find resources by name or keyword
- **Mobile-friendly** — responsive design that works on any device
- **Accessible** — clear typography, high contrast crisis banner, tap-to-call phone numbers

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and import the repository
3. Vercel will auto-detect Vite and deploy — no configuration needed

## Tech Stack

- React 18
- Vite 5
- No external UI libraries — pure CSS-in-JS for zero dependencies

## Important Note

This directory is for informational purposes and may not include every available service. For the most comprehensive and current listings, visit [HelpStartsHere.gov.bc.ca](https://www.helpstartshere.gov.bc.ca).

**If you are in immediate danger, please call 911.**
